The bachelor course consisted of lectures, exercises and a python programming project dealing with fundamental Machine Learning methods.

## Structure
* Summer term 2026
* Teaching assignment from University of Lübeck
* German lecture with exercises and python programming project

## Content
* Fundamental Machine Learning Methods
* Classification, Regression
* (un-)informed and adversarial Search 
* Hillclimbing, Simulated annealing 
* Decision Trees
* Naive Bayes
* Support Vector Machine
* Linear, Extended and Unscented Kalman Filter
* Neural Networks
* Applications
